Cube
Description

My first project with a 3D object using ThreeJS and Blender.

Background

I wanted to create a project featuring an interactive 3D object that users could engage with. However, loading a 3D object onto a website turned out to be more challenging than I anticipated. For now, I’ll upload this project as it is, using it as a foundation for future projects that can build upon this one.

I find working with 3D projects both new and refreshing, and I have many ideas for more complex 3D creations. While this project is quite simple on its own, it serves as a stepping stone for developing more advanced 3D projects in the future.

Features
  • Mouse left/right click: Move the camera / object
  • Mouse wheel (click/scroll): Zoom in/out
Future Improvements
  • Add texture
  • Better lighting
  • Better background/scene
  • More interactive features
