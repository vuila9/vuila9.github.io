Bejeweled 2
Open as app ↗

Click or tap a gem, then an adjacent gem, to swap them. Match 3 or more in a row or column to clear them. On mobile, open the app and use your browser's “Add to Home Screen” for true fullscreen.

Description

A faithful web revival of the classic Bejeweled 2. The artwork and sound effects are the originals, extracted from the Android release; the game itself is re-implemented from scratch in vanilla JavaScript on an HTML5 canvas so it runs in any modern browser — including on a phone, a tablet, where it can be added to the home screen and played fullscreen.

Background

The game was made with the assisted of Claude AI.

After successfully reviving Flappy Bird as a web app, I wanted to try the same approach on a different genre — a match-3 puzzle game. Bejeweled 2 was the obvious pick: it is the game that defined the entire match-3 category, it works completely offline as a single-player title, and an old standalone Android APK is preserved on the internet, which means all of its original assets are available to extract.

The process mirrors the Flappy Bird project. I pulled the original sprites and sound effects from the APK, then re-implemented the game logic — the gem grid, swapping, match detection, cascading, and scoring — from scratch in vanilla JavaScript on an HTML5 canvas. Because the game is not a live-service title, a static website hosted on my GitHub Pages portfolio is all it needs.

As with Flappy Bird, the goal was to make it feel like a native app. Safari lets you add a web app to your Home Screen and launch it as a standalone, full-screen application, and once it has loaded the first time it can be played entirely offline. See the setup instructions below to achieve this.

iOS App Setup Instruction

Quick disclaimer: Android users can always get the vanilla game by installing the APK file, so yes you are being left out on this one.

Before we get started, open the webapp version on Safari browser, because apparently only Safari can do this.

The version of the game on this current page is not a webapp version, make sure you only do the next step once you open the webapp version of the game on a new tab.

Features
  • Original assets: gem sprites and sound effects pulled straight from the Android APK.
  • Swap (click / tap): select a gem and an adjacent gem to swap them and make a match.
  • Cascades: chained matches from falling gems award bonus points.
  • Levels: a progress meter fills as you clear gems; filling it advances the level, awards a bonus, and raises the next target.
  • Score & best: local best score persists between sessions.
  • Fullscreen (button): expand the game to fill the screen.
  • Esc (keyboard): exit fullscreen.
  • Offline playable: Once loaded up the first time, the game can be fully played without internet. (only applied after adding the game as a native app)
Why you can play offline after the first time?

The first time you open the game with internet, your mobile device quietly saves a complete copy of the game onto the device - all the pictures, sounds, and the game itself. Adding it to your Home Screen turns that saved copy into an app icon, just like a real app.

After that, whenever you tap the icon, the game runs from the copy already on your phone instead of downloading anything. That's why it opens instantly and works perfectly even in airplane mode or with no signal.

Occasionally, I will come back and update the game, if connected to internet, the game will be automatically update to a newer version seamlessly without needing to redo the setup steps.
