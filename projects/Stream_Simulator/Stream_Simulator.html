Description

This is a streaming simulator inspired by the Twitch platform, built entirely using HTML, JavaScript, and CSS. I aimed to replicate the real-life streaming experience as realistically as possible. Expect features like viewer interactions through chats and reactions, random gifting/subscriptions, fluctuating viewer counts, and broadcaster privileges such as banning, modding, etc. and many more.

Please watch the attached YT video above for a full walkthrough of the program, or type '/command' into the chat to show a list of all supported commands. More info are available in the Features section.

Features

The majority of these features scales with total viewer count. With a high viewer count, the frequency of happening will increase, and vice versa. The duration of certain events also scales with viewer count. With a high viewer count, the duration is extended, and vice versa. The stream will remain running indefinitely once started. Some features will only happen when chat is running. Some feature will pause chat when applied.

Features that scales with viewer count:

  • Viewer count: Viewer count will fluctuate up and down. Fluctuation rate scales with view count.
  • Chat output: Chat will be generated indefinitely. Chat output rate scales with view count.
  • Subscription/Gifting: Random viewers will subscribe or gift random amount of subs to the chat. The chance of this scales with view count.

Chat reaction (Duration and frequence scales with viewer count):

  • Greeting: chat will greet you when the stream just started.
  • Gifting: chat will react based on the amount of gifted subs being generated.
  • Ban/Unban/Mod/Unmod: chat will react when a viewer gets banned or unbanned (by you).

Commands (also displayed when enter '/command'):

  • /username name: change your user name
  • /title title: change stream title
  • /category category: change stream category
  • /mod name/@name: make a viewer a moderator
  • /unmod name/@name: demote a moderator
  • /vip name/@name: make a viewer a vip
  • /unvip name/@name: demote a vip
  • /founder name/@name: make a viewer a founder
  • /ban name/@name: ban any viewer
  • /unban name/@name: unban any banned viewer
  • /gift name/@name: gift a sub to a viewer
  • /giftrandom number: gifting 1 - 100 subs to random viewers
  • /giftrate 1-10: increase usual gifted sub by 1-10 times
  • /giftlog: show log of all gifter
  • /spam word/"phrase" or both: trigger the chat to spam specified word(s) and/or phrase(s)
  • /viewcount number: set viewer count to any number from 1 - 999,999
  • /yt URL: change the embedded YouTube video
  • /clearpopup: remove all user profile popups
  • /clear: clear chat
  • /mute: mute all sub/prime/gift sound alert
  • /suboverlay on/off: turn sub overlay on/off
  • /screencapture: quick disclaimer about this feature

Buttons feature:

  • Start/Pause Chat (button): Start/pause/resume chat.
  • Screen Capture (button): Disabled by default. Capture user's screen content. To see disclaimer: enter '/screencapture' in the chat.
  • Arrow key up/down: traversing chat history.
  • Chat (button/enter): output your message onto the chat.
  • Follow/Subscribe (button): does nothing, for decoration.
